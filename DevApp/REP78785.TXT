    PROCEDURE ModifyOrCreateTempItemLine@1000000004(VendorCode@1000000000 : Code[20];ItemCode@1000000001 : Code[20];Quant@1000000002 : Decimal;PartQuant@1000000007 : Decimal;Amnt@1000000003 : Decimal;Desc@1000000004 : Text[200];Uom@1000000005 : Code[20];Negeer@1000000006 : Boolean;Verkooppr@1000000008 : Decimal);
    BEGIN
      TempItemLines.RESET;
      TempItemLines.SETRANGE(Vendor,VendorCode);
      TempItemLines.SETRANGE("Item Code", ItemCode);
      IF TempItemLines.FINDFIRST THEN BEGIN
         IF Negeer THEN
           TempItemLines."Qty. per Unit of Measure" += Quant
         ELSE
           TempItemLines."Qty. per Unit of Measure" += Quant * PartQuant;
         IF Negeer THEN
           TempItemLines."Nett Price" += Quant * Amnt
         ELSE
           TempItemLines."Nett Price" += (Quant * PartQuant * Amnt);
         //Toevoegen verkoopprijs indien aanwezig
         TempItemLines."Gross Price" += (Verkooppr * Quant);
         TempItemLines.MODIFY;
      END ELSE BEGIN
        TempItemLines.INIT;
        TempItemLines.Vendor := VendorCode;
        TempItemLines."Item Code" := ItemCode;
        IF Negeer THEN
          TempItemLines."Qty. per Unit of Measure" := Quant
        ELSE
          TempItemLines."Qty. per Unit of Measure" := Quant * PartQuant;
        IF Negeer THEN
          TempItemLines."Nett Price" := Quant * Amnt
        ELSE
          TempItemLines."Nett Price" := (Quant * PartQuant * Amnt);
        //Toevoegen verkoopprijs indien aanwezig
        TempItemLines."Gross Price" += (Verkooppr * Quant);
        TempItemLines."Gross Weight" := Amnt;
        TempItemLines."Application Unit" := Uom;
        IF STRLEN(Desc) > 50 THEN
          TempItemLines."Item Description" := COPYSTR(Desc, 1, 50)
        ELSE
          TempItemLines."Item Description" := Desc;
        TempItemLines.INSERT;
      END;
      TempItemLines.RESET;

      IF NOT TempVendorRec.GET(VendorCode) THEN BEGIN
        TempVendorRec.INIT;
        TempVendorRec."No." := VendorCode;
        IF VendorCode <> '' THEN BEGIN
          IF VendRec.GET(VendorCode) THEN
            TempVendorRec.Name := VendRec.Name;
          IF Vend1Rec.GET(VendorCode) THEN
            TempVendorRec.Name := Vend1Rec.Name;
        END ELSE
          TempVendorRec.Name := 'Leeg';
        TempVendorRec.INSERT;
      END;
    END;

        IF Negeer THEN

        TempSubcLines.MODIFY;
      END ELSE BEGIN
        TempSubcLines.INIT;
        TempSubcLines.Vendor := Subc;
        TempSubcLines."Item Description" := COPYSTR(Desc, 1, 50);
        TempSubcLines."Item Code" := Geti;
        IF Negeer THEN
          TempSubcLines."Qty. per Unit of Measure" := Quant
        ELSE
          TempSubcLines."Qty. per Unit of Measure" := Quant * PartQuant;
        IF Negeer THEN
          TempSubcLines."Nett Price" := Quant * Price
        ELSE
          TempSubcLines."Nett Price" := Quant * Price * PartQuant;
        TempSubcLines."Gross Weight" := Price;
        TempSubcLines."Application Unit" := UOM;
        TempSubcLines.INSERT;
      END;
